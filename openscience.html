Open Science

Sharing code, knowledge and data openly are fundamental to the progress of science.
In that spirit, I am involved in the development of many open source toolboxes.

I am a core contributor to the following software packages:
  • MNE-Python: a software package for MEG and EEG data analysis.
  • HNN-core: a software package for simulation and optimization of neural circuits underlying MEG/EEG source estimates.
  • MNE-BIDS: a package for facilitating data exchange and analysis with the Brain Imaging Data Structure (BIDS), a standard that I co-developed for MEG.
  • pyglmnet: a Python package to fit regularized general linear models.
